Aan de slag

De makkelijkste manieren om op Hashiverse te posten zonder veel code te schrijven. Kies degene die bij jou past.

Deelknoppen op je website

Zet een knop op je blog of site die een Hashiverse-conceptbericht opent dat is voorgevuld met een link en tekst. De href van de knop wijst rechtstreeks naar de Hashiverse-app: 

https://app.hashiverse.com/#/share?text=YOUR_TEXT&url=YOUR_URL

Beide query-parameters zijn optioneel en moeten URL-gecodeerd zijn. Hieronder twee starter-knopstijlen — elke snippet is volledig op zichzelf staand: plak hem zoals hij is en de knop werkt. Vervang YOUR_TEXT en YOUR_URL door de link en het bericht die je vooraf wilt invullen.

Posten vanuit Python

Installeer de Python-client vanaf PyPI:

pip install hashiverse-client

Post dan in vijf regels:

from hashiverse_client import HashiverseClient

client = HashiverseClient.create_from_keyphrase(
    key_phrase="my secret keyphrase",
    data_dir="~/.hashiverse",
)
client.post("Hello from Python! #hashiverse-python")

Zie de volledige Python-client op GitHub voor geavanceerde functies — tijdlijn-reads, hashtag-streams, sleutelbeheer en opnieuw verbinden vanuit een opgeslagen sleutel.
