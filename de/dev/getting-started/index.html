Erste Schritte

Die einfachsten Wege, in Hashiverse zu posten, ohne viel Code zu schreiben. Wähle den, der zu dir passt.

Share-Buttons auf deiner Website

Setze einen Button auf deinem Blog oder deiner Website, der einen Hashiverse-Compose-Entwurf öffnet, der mit einem Link und Text vorgefüllt ist. Das href des Buttons zeigt direkt auf die Hashiverse-App: 

https://app.hashiverse.com/#/share?text=YOUR_TEXT&url=YOUR_URL

Beide Query-Parameter sind optional und sollten URL-kodiert sein. Unten zwei Starter-Button-Stile — jedes Snippet ist vollständig eigenständig: einfach einfügen und der Button funktioniert. Ersetze YOUR_TEXT und YOUR_URL durch den Link und die Nachricht, die du vorgefüllt haben möchtest.

Posten aus Python

Installiere den Python-Client von PyPI:

pip install hashiverse-client

Dann poste in fünf Zeilen:

from hashiverse_client import HashiverseClient

client = HashiverseClient.create_from_keyphrase(
    key_phrase="my secret keyphrase",
    data_dir="~/.hashiverse",
)
client.post("Hello from Python! #hashiverse-python")

Sieh dir den vollständigen Python-Client auf GitHub an für erweiterte Funktionen — Timeline-Reads, Hashtag-Streams, Schlüsselverwaltung und das Wiederverbinden über einen gespeicherten Schlüssel.
