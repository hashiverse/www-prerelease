Per-PayloadRequestKind running counter of inbound dispatches. Incremented after packet decode + replay-guard but before per-handler PoW gates, so adversarial load shows up too.

§peer_stats_response_cache: Mutex<Option<(TimeMillis, PeerStatsResponseV1)>>

Cached signed stats blob. The TimeMillis is the timestamp recorded in the cached PeerStatsResponseV1 itself — the cache hands the response back verbatim so clients re-sharing it get a single canonical byte sequence.
