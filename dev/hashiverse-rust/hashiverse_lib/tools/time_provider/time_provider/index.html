§TimeProvider trait — the clock seam

A small trait every component takes as an Arc<dyn TimeProvider> instead of calling SystemTime::now() or tokio::time::sleep directly. RealTimeProvider delegates to wall-clock time; crate::tools::time_provider::manual_time_provider lets tests advance time by hand and makes every sleep return immediately, so deterministic integration tests can span simulated days in milliseconds of real wall time.
